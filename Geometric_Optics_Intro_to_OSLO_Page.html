Introduction to OSLO¶

OSLO requires a complete description of the optical system through a set of parameters which are organized into the Surface Data Spreadsheet (Lens > Surface Data Spreadsheet).

Each row of the table describes a different surface and each column within represents a specific property of that surface. Each column headers has a precise meaning. It is important that the correct values are entered into OSLO in order for the results to best match those of the desired optical system. It is thus imperative that steps be taken to properly determine and verify the correct inputs for the columns. The suggested series of steps to follow for those just getting started with OSLO are,

STEPS

  1. Sketch the Optical System (on paper!)

  2. Find the Vertex for each surface

  3. Find the Curvature of each surface

  4. Find the Distance to next surface

  5. Find the Height of each surface

  6. Find the Material to next surface

  7. Tabulate the result

Additional steps may also be needed such as,

  1. Setting the index of refraction

  2. Changing the object size

  3. Forcing OSLO to draw surface(s)

  4. Changing the rays that are drawn

1. Sketch the optical system¶

The first step in understanding OSLO will be to understand the numbers and values it is asking for. To do this, sketch out a simple object-lens-image diagram similar to what is shown below on paper.

2. Find the Vertex for each surface¶

Start off by identifying each vertex in the surface, and marking each with a vertical line where each surface crosses the optical axis. These vertical lines will be used to gauge distances from each surface.

Label the object on the left with “OBJ”, and the image on the right with “IMS”. Normally one would label the left side of the lens with a “1” and the right side of the lens with a “2”.

However, OSLO does not follow the normal labelling convention. Instead, one surface, either “1” or “2” will always be the aperature stop, or “AST”. In this system, the left side of the lens, what normally would’ve been labelled as “1”, will be labeled “AST” instead. The other keeps the conventional label of “2.”

3. Find the curvature of each surface¶

Using the Lens Maker’s formula at the beginning of this section, the radius of curvature \(r\) for a bi-convex lens, assuming both surfaces have the same curvature (\(r=|r_1|=|r_2|\)), is

\[r = \frac{f}{2(n-1)}\]

You should verify this result!

To force OSLO to show a surface, one uses the button in the SPECIAL column. In the row corresponding to the surface that needs to be drawn, click the button under the SPECIAL column and navigate to Surface Control (F) -> General to get the following dialog box
