More with OSLO¶

Aspherical surface¶

An aspherical surface profile can reduce or eliminate spherical aberration and also reduce other optical aberrations such as astigmatism, compared to a simple lens. Alternatively, an aspherical profile maybe needed to better approximate more complex surface profiles.

Adding such a correction to a surface can be done by using the SPECIAL column. Click the button and select Polynomial Asphere (A) -> Conic/Toric to bring up the following dialog box,

One may choose from one of 3 surface types, Rotationally symmetric, Cylinder, and Toroid.

Slider Wheel¶

When attempting to optimize a multi-lens system in OSLO it is benificial to use the slider wheel. This eliminates the need to type numbers repeatedly into a spreadsheet, and gives better control on seeing when the system is optimized. The “slider-wheel” feature of OSLO creates a linear slider control that changes a variable (“wheel” is in reference to the fact that the mouse’s scroll-wheel can be used to change the value.)

To create a slider-wheel, go to the optimize menu, select Slider-Wheel Design and fill in the desired elements. You may also choose to display aberration curves such as Optical Path Difference (OPD) or ray error (Ray-intercept). You can read more on these aberration curves in Chapter 33 of the Handbook for Optics. If no such curves are desired, you can of course select Draw only.
