Activities (Week 1)¶

In the following lab activities, you will be dealing with optical systems involving a single bi-convex lens.

IMPORTANT

Students should show results of each task completed in the following activities (provide OSLO drawing and spreadsheet in the case of this Activity 3).

Activity 1¶

You are faced with the following problem:

You need to show a tiny graph of your previous experimental results to your nearly blind instructor. The graph has a size of \(10\ mm\) and for the instructor to see any details he needs the graph to be \(100\ mm\) in size. Using a bi-convex lens with a focal length of \(50\ mm\), perform the following tasks,

Tasks¶

  1. Determine the distance away from the graph that the lens must be in order to acheive this magnification.

  2. How far away must the “blind” instructor be from the graph to see the magnified picture clearly?

  3. Use the physlet simulation below (or open physlet in separate tab) to verify the results.

Physlet simulation¶

Activity 2¶

In this activity you will design a \(1:1\) imaging setup using a bi-convex lens with the following properties:

  • The focal length of the lens is \(50\ mm\)

  • The index of refraction of the lens is \(1.5\)

  • The object is \(100\ mm\) from the lens.

  • The object height is \(15\ mm\)

  • The lens diameter is \(38\ mm\)

  • The lens thickness is \(8\ mm\)

Tasks¶

  1. On a piece of paper calculate the position of the image. What is the image distance?

  2. On a piece of (ideally graph paper), draw the setup including the positions of the object, lens, and its focal length using the three principal rays.

Activity 3¶

In this activity you will design a \(1:1\) imaging setup using OSLO consisting of a bi-convex lens with the same properties as in the Activity I:

  • The focal length of the lens is \(50\ mm\)

  • The index of refraction of the lens is \(1.5\)

  • The object is \(100\ mm\) from the lens.

  • The object height is \(15\ mm\)

  • The lens diameter is \(38\ mm\)

  • The lens thickness is \(8\ mm\)

Tasks¶

  1. Determine the surfaces OBJ, AST, 2, and IMS of a single lens setup (including screen).

  2. After setting the index, determine RADIUS, THICKNESS, APERTURE RADIUS, and GLASS values for each surface.

  3. Draw the optical system by clicking the DRAW OFF button. What are the problems with the result? How does it differ from the ray tracing diagrams from the previous Activities? What is the focal length and does it match the desired value?

  4. Obtain the correct focal length by adjusting the curvature of the appropriate surface. Make sure to maintain equal curvature of both sides of the convex lens. You can do so more easily using the following tip:

    5. OSLO TIP - Dependent parameters

    Because this lens is equi-convex, it is useful to set one of the radii of curvatures to automatically be the negative of the other surface radius of curvature. To do this, click on the button in the RADIUS column of the row numbered 2. Select Minus Curvature Pickup and answer 1 when asked to Enter the pickup source surface and then click OK for the next two dialog boxes. Now when you change the RADIUS of first surface (the AST surface), the RADIUS of the second surface will automatically get updated!

  5. Are there any important surfaces missing from the ray tracing diagram (that is drawn by OSLO)? If so which ones? Force OSLO to drawn all the missing surfaces if there are any.

  6. Correct the Object height if it is not the desired value.

  7. Adjust the height of the source point for the rays (on the object) to what is most appropriate for comparision with Activity 2. You can do so using the following tip:

    8. OSLO TIP - Using more of the lens

    With the default values, usually only the most central rays are shown. How does one trace rays through the top and bottom of the lens? The trick here is to change the Ent beam radius (entering beam radius) to the desired height (This is the radius for a ray in the aperture plane).

  8. After making all the necessary corrections, verify that all the requirements are met. If additional adjustments are desired, refer to the OSLO User Guide for more info.

  9. Provide an analysis of the results. Are there any remaining issues? Do all the rays intersect at the same point? When the height of the source point (Ent beam radius) of the rays is changed, do any other important properties change as well? Compare to the sketch in Activity 1. Provide a detailed discussion.
